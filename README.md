# QRPix

<img src="./static/logo.png" width="200" />

- Simple web application that allows generating QR Codes for PIX payments

## Requirements

- Golang 1.24.1 or higher  
   - [Download](https://go.dev/dl/)

## Build and Run

```bash	
go build
```

- Linux/Mac
    ```bash	
    ./qrpix
    ```
- Windows
    ```bash
    ./qrpix.exe
    ```
